# Enterprise Management System 🏢

Sistema integral de gestión empresarial desarrollado como Trabajo Integrador para la materia Programación 1. Esta solución utiliza una arquitectura de capas para gestionar operaciones críticas de una organización, desde el control de inventario hasta la logística.

## 🚀 Tecnologías
* **Backend:** .NET 8 (Web API)
* **Documentación:** Swagger / OpenAPI
* **Arquitectura:** Capas (Entities, DTOs, Service, Data, WebApi)
* **Pruebas:** NUnit para Testing Automatizado

## 🛠️ Módulos Principales
* **Gestión de Clientes:** CRUD completo y administración de base de clientes.
* **Control de Stock:** Sistema de actualización de productos con alertas automáticas de bajo stock.
* **Módulo de Compras:** Registro y seguimiento de transacciones comerciales.
* **Logística y Viajes:** Gestión de flota (camionetas) y seguimiento de rutas/viajes.

## 🧪 Calidad de Software
El proyecto incluye un conjunto de pruebas unitarias que validan la lógica de negocio en cada módulo (Clientes, Compras, Productos y Viajes), asegurando la integridad del sistema ante cambios.
